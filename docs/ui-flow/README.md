# UI Flow

Thư mục này dùng để lưu tài liệu/mapping liên quan FE prototype.

## Trạng thái hiện tại

Backend đã có API đủ để FE dựng demo Flow 1:

```text
Customer lookup → Loan application draft → Asset → Valuation → Eligibility → Submit for approval
```
